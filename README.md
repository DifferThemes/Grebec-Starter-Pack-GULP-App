### Run Server

Set **PHPserver** to true if you need proxy server
```js
const PHPserver = false;
const domain = 'localhost/gulp/app';
const root = 'app';
```

```bash
npm i
gulp
```

### Just Compile .SCSS

```bash
gulp scss
```

### Just Compress .JS

```bash
gulp js
```

### Compress images

```bash
gulp img
```
